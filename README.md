# Word Counter

Count words, characters, sentences, and estimated reading time in your texts.

## Live Demo

[View Live Demo](https://w0rd-counter.netlify.app/)

## Features

- Real-time word counting
- Character counting (with and without spaces)
- Sentence and paragraph counting
- Reading time estimation
- Keyword density analysis
- Character limit (10,000)
- Export text to file
- Copy to clipboard
- Clear all text
- Full Unicode support (all languages)
